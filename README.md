# GraphCare Website

Landing page for GraphCare knowledge extraction service.

## Deployment

**Domain:** graphcare.mindprotocol.ai

**Tech Stack:**
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Deployed to Vercel

## Development

```bash
npm run dev
# Open http://localhost:3002
```

## Build

```bash
npm run build
npm start
```

## Deploy

```bash
vercel deploy --prod
```
